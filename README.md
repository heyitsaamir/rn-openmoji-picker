# rn-openmoji-picker

A picker for react native backed by the openmoji project

## Installation

```sh
npm install rn-openmoji-picker
```

## Usage

```js
import RnOpenmojiPicker from "rn-openmoji-picker";

// ...

const result = await RnOpenmojiPicker.multiply(3, 7);
```
